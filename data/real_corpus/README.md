# Real Corpus Directory

## 1. Purpose
This directory is reserved for future EduCode-1.5B small real-data training work.

## 2. Current Status
- This directory currently contains structure only.
- No real corpus has been added yet.
- No external dataset has been downloaded.
- No private notes or copyrighted material should be added without explicit approval.

## 3. Directory Layout

raw/
- future raw allowed source files
- not committed by default if large/private

processed/
- cleaned text/jsonl outputs
- generated by future data pipeline

splits/
- train/val split files
- generated after data intake

metadata/
- source/license/intake metadata

## 4. Allowed Source Types
Allowed:
- user-created notes explicitly approved for this project
- permissively licensed educational text
- public domain text
- small permissively licensed code snippets
- manually curated local examples

## 5. Disallowed Source Types
Disallowed:
- private credentials
- API keys
- private personal data
- copyrighted books/PDFs/course material without permission
- large web scrapes
- unclear-license datasets
- private chats unless explicitly approved

## 6. Git Policy
- raw and processed large data should not be committed by default
- metadata and small templates may be committed
- generated large files should stay ignored
- before making repo public, re-check this directory
