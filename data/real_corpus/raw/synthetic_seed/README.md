# Synthetic Seed Corpus

This directory contains small project-authored synthetic educational examples.

Purpose:
- provide a safe seed corpus for future intake, cleaning, tokenizer, and small data-pipeline validation
- avoid external downloads and copyright ambiguity
- stay explicitly separate from real-world external corpora

Current rules:
- all files here are synthetic examples written for this project
- no secrets, private data, or copied external material should appear here
- this directory is intentionally small and should not be described as real pretraining data
