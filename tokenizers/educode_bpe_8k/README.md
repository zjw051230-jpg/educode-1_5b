# EduCode BPE 8k

This directory stores the `educode_bpe_8k` tokenizer artifact trained only on the processed synthetic seed corpus.
It is a formal-path smoke artifact for tokenizer integration work.
It is not a production tokenizer and does not imply real-data readiness.

Observed vocab size: 1174
Target vocab size: 8192
Corpus path: `data/real_corpus/processed/synthetic_seed.processed.jsonl`
Train split used: `train` only
