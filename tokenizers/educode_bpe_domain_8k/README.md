# EduCode Domain BPE 8k

This directory stores the `educode_bpe_domain_8k` tokenizer artifact trained only on the processed 45-file synthetic expanded corpus.
It is a tokenizer-only artifact for later data/model smoke work.
It is not a production tokenizer and does not imply real-data readiness.

Observed vocab size: 3988
Target vocab size: 8192
Corpus path: `data/real_corpus/processed/synthetic_expanded.processed.jsonl`
Train split used: `train` only
